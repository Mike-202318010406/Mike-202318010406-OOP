##Integrated Management System Project

##Project Overview
This project is an Integrated Management System (IMS) that consolidates four distinct Java subsystems into a cohesive application. The subsystems include Zoo Management, Banking Task Management, Restaurant Management, and Shape Parsing & Analysis. The project emphasizes practical experience with Java OOP concepts and introduces basic system integration and GUI development.

##Building the Project
To build this project, ensure all the Java files are organized in their respective directories as shown in the file structure. You can compile the project using a Java compiler.
javac BankingTaskManagement/*.java
javac RestaurantManagement/*.java
javac ShapeAnimation/*.java
javac ZooManagement/*.java

##Running the Project
After building the project, you can run the main classes for each subsystem.
For the Banking Task Management System:
java BankingTaskManagement.Bankingtasklistgui
For the Restaurant Management System:
java RestaurantManagement.RestaurantManagementSystem
For the Shape Animation System:
java ShapeAnimation.Main
For the Shape ZooManagement System:
java ZooManagement.Main

##Testing
Each subsystem has its own set of functionalities that can be tested individually. For comprehensive testing, you can create test cases for each class to ensure they behave as expected.
